I was just moderating the asp.net forums and approved Phil's announcement 29 minutes ago that Asp.net MVC 2.0 RC 2 has just been released:

Hi everyone,

We released ASP.NET MVC 2 Release Candidate 2 this evening!

  • Download: http://go.microsoft.com/fwlink/?LinkID=182483
  • Release notes: http://go.microsoft.com/fwlink/?LinkID=182482
  • Announcement blog post: http://haacked.com/archive/2010/02/04/aspnetmvc2-rc2.aspx
  • ASP.NET MVC FAQ: http://blogs.msdn.com/rickandy/archive/2009/10/15/mvc-faq.aspx
  • Documentation: http://msdn.microsoft.com/en-us/library/dd394709(VS.100).aspx

This release works with Visual Studio 2008 SP1 and Visual Web Developer 2008 SP1. We will be posting source code and our futures assembly soon.

Thanks,

The ASP.NET Team

Phil Haack (http://haacked.com/)
